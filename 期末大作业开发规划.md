工业故障诊断 AI Agent 开发规划与实施指南本项目旨在基于 LangGraph 构建一个具有“人机协同 (Human-in-the-loop)”特征的工业故障诊断 Agent。Agent 将通过内外双擎检索（本地 Milvus + 外部 Tavily）获取信息，经 LLM 过滤融合后生成初始流程，并自主审计缺口、结合专家反馈，最终输出一份严谨的工业故障处理 Mermaid 流程图。项目进阶大满贯亮点：RAG + Web Search 双擎并发 (满足 Concurrency 要求)。集成四大 MessageState (System, Human, AI, Tool)。包含中断与人机循环 (满足 Loop 要求)。接入 Langfuse 实现全链路可观测性。第一阶段：环境与基础准备 (预计 1-2 小时)1.1 环境搭建除了之前的依赖，新增 tavily-python：pip install

第二阶段：LangGraph 核心架构设计 (核心升级)2.1 定义 State (状态)状态字典需要扩容，用来容纳内外部不同的知识。from typing import Annotated, TypedDict, List
import operator
from langchain_core.messages import BaseMessage
from langgraph.graph.message import add_messages

class AgentState(TypedDict):
    messages: Annotated[List[BaseMessage], add_messages]
    fault_input: str             
    internal_knowledge: str      # Milvus 查到的内部 SOP
    external_knowledge: str      # Tavily 查到的外部资讯
    filtered_context: str        # LLM 过滤融合后的最终上下文
    mermaid_diagram: str         
    audit_questionnaire: str     
    has_gaps: bool               
2.2 构建功能 Node (包含并发与过滤)Node 1: retrieve_internal_node从 Milvus 获取工厂内部 SOP。Node 2: retrieve_external_node调用 tavily_tool 搜索，比如“某型号电机过热最新维修通报”，将结果作为 ToolMessage 存入 messages，并提取文本存入 external_knowledge。Node 3: filter_knowledge_node (知识过滤与融合)核心逻辑：LLM 读取 internal_knowledge 和 external_knowledge。Prompt 要求：“你是一个资深工程师。请剔除外部搜索中的广告和无关信息，将内部标准 SOP 和外部最新情报融合，输出一份精炼的故障诊断背景上下文。” 结果存入 filtered_context。Node 4: generate_diagram_node基于 filtered_context 生成 Mermaid 流程图。Node 5: audit_diagram_node审计图纸是否存在缺口，生成问卷，更新 has_gaps。Node 6: human_expert_node人类反馈输入。Node 7: refine_diagram_node结合人类反馈重绘。2.3 构建图与控制流 (包含 Concurrency 与 Loop)from langgraph.graph import StateGraph, END

# 这里简化为：入口同时分发给内外检索节点（通过一个统一的入口 Node 或者直接配置两个起始条件）
# 为了逻辑清晰，我们假设一个虚拟起点分发：
def dispatch(state): pass
workflow.add_node("start_dispatch", dispatch)
workflow.set_entry_point("start_dispatch")

workflow.add_edge("start_dispatch", "retrieve_internal")
workflow.add_edge("start_dispatch", "retrieve_external")

# 内部和外部检索完成后，汇聚到过滤节点 (Fan-in)
workflow.add_edge(["retrieve_internal", "retrieve_external"], "filter")
# --------------------------------------

workflow.add_edge("filter", "generate")
workflow.add_edge("generate", "audit")

# --- 构建逻辑循环 (Loop Edge) ---
def check_gaps(state: AgentState):
    if state.get("has_gaps"): return "needs_feedback"
    return "finished"

workflow.add_conditional_edges(
    "audit", check_gaps, {"needs_feedback": "human", "finished": END}
)

workflow.add_edge("human", "refine")
workflow.add_edge("refine", "audit") # 闭环回到审计

app = workflow.compile()
